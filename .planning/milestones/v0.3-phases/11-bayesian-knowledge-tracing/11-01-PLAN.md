BKT equations from .planning/06-spaced-repetition.md:
```
After correct:   P(L|correct)   = P(L) * (1 - P(S)) / (P(L) * (1 - P(S)) + (1 - P(L)) * P(G))
After incorrect: P(L|incorrect) = P(L) * P(S)       / (P(L) * P(S)       + (1 - P(L)) * (1 - P(G)))
After observation: P(L_new) = P(L|obs) + (1 - P(L|obs)) * P(T)

Defaults: P(L0)=0.1, P(T)=0.3, P(S)=0.1, P(G)=0.25

Age adjustments:
| Param | Age 6-7 | Age 7-8 | Age 8-9 |
| P(G)  | 0.30    | 0.25    | 0.20    |
| P(S)  | 0.15    | 0.10    | 0.08    |
| P(T)  | 0.25    | 0.30    | 0.35    |

Thresholds: P(L) >= 0.95 = mastered, P(L) < 0.40 = re-teaching
```

```typescript
export interface BktParams {
  /** Prior knowledge probability P(L0) */
  pL0: number;
  /** Learn rate P(T) — probability of learning per opportunity */
  pT: number;
  /** Slip rate P(S) — probability of wrong answer despite knowing */
  pS: number;
  /** Guess rate P(G) — probability of right answer without knowing */
  pG: number;
}

export interface BktUpdateResult {
  /** Updated mastery probability P(L) after this observation */
  newPL: number;
  /** Whether skill meets mastery threshold (>= 0.95) */
  isMastered: boolean;
  /** Whether skill needs re-teaching (< 0.40) */
  needsReteaching: boolean;
}
```

Create `src/services/adaptive/bktCalculator.ts` with:

- `DEFAULT_BKT_PARAMS: BktParams` with research defaults (pL0=0.1, pT=0.3, pS=0.1, pG=0.25)
- Age bracket map (constant, not computed):
  - Ages 6-7 (childAge 6 or 7): { pL0: 0.1, pT: 0.25, pS: 0.15, pG: 0.30 }
  - Ages 7-8 (childAge 8): { pL0: 0.1, pT: 0.30, pS: 0.10, pG: 0.25 }
  - Ages 8-9 (childAge 9): { pL0: 0.1, pT: 0.35, pS: 0.08, pG: 0.20 }

  NOTE on age bracket mapping: The brackets in the research doc are labeled "Age 6-7", "Age 7-8", "Age 8-9". Map these by `childAge` integer value:
  - childAge 6 or 7 → "Age 6-7" params
  - childAge 8 → "Age 7-8" params (8-year-olds are in the middle bracket)
  - childAge 9 → "Age 8-9" params
  - childAge null/undefined → DEFAULT_BKT_PARAMS (research defaults, which match age 7-8)

- `getBktParams(childAge: number | null): BktParams` — pure function, returns params for age bracket

  1. Compute P(L|observation) using the correct/incorrect formula
  2. Apply learning: P(L_new) = P(L|obs) + (1 - P(L|obs)) * P(T)
  3. Return { newPL, isMastered: newPL >= 0.95, needsReteaching: newPL < 0.40 }

Create `src/services/gamification/weeklyStreak.ts` with these pure functions:

1. `getISOWeekNumber(date: Date): { year: number; week: number }` — Returns the ISO 8601 week number (Monday-based) and year. Use the standard algorithm: find the Thursday of the date's week, derive the year and week from that Thursday. Return both year and week to handle year boundaries correctly (e.g., Jan 1 can be in week 52 of prior year).

2. `isSameISOWeek(dateA: Date, dateB: Date): boolean` — Returns true if both dates fall in the same ISO week. Compares year AND week number.

3. `isConsecutiveWeek(earlier: Date, later: Date): boolean` — Returns true if `later` is in the ISO week immediately following `earlier`. Handles year boundaries (week 52/53 -> week 1).

4. `computeStreakUpdate(lastSessionDate: string | null, currentDate: Date, currentStreak: number): { newStreak: number; practicedThisWeek: boolean }` — The core logic:
   - If `lastSessionDate` is null (first-ever session): return `{ newStreak: 1, practicedThisWeek: true }`.
   - Parse `lastSessionDate` as Date.
   - If same ISO week as currentDate: return `{ newStreak: currentStreak, practicedThisWeek: true }` (no change, already counted).
   - If consecutive ISO week: return `{ newStreak: currentStreak + 1, practicedThisWeek: true }` (streak continues).
   - If more than one week gap: return `{ newStreak: 1, practicedThisWeek: true }` (streak resets, but this session starts fresh at 1).

Note on reset logic per CONTEXT.md: "Full missed week resets streak to 0." But since we are IN a session completing right now, we start a new streak of 1 (not 0). Streak 0 means "never practiced" or "display state before any session this week." The computeStreakUpdate function returns the streak AFTER this session commits.

Update `src/services/gamification/index.ts` barrel export to include all weeklyStreak exports.

Create `src/__tests__/gamification/weeklyStreak.test.ts` with tests:

**getISOWeekNumber:**
- Monday 2026-03-02 is in week 10 of 2026
- Sunday 2026-03-08 is also in week 10 of 2026 (same week)
- Monday 2026-03-09 is in week 11 of 2026 (next week)
- Year boundary: 2025-12-29 (Monday) is in week 1 of 2026 (ISO week rule)

**isSameISOWeek:**
- Same week returns true (Mon and Sun of same week)
- Different weeks return false

**isConsecutiveWeek:**
- Week 10 to week 11 returns true
- Week 10 to week 12 returns false
- Year boundary: last week of year to week 1 returns true

**computeStreakUpdate:**
- First session (null lastSessionDate): streak becomes 1
- Same week session: streak unchanged
- Consecutive week: streak increments by 1
- Two-week gap: streak resets to 1 (new streak started)
- Three-week gap: streak resets to 1
- Multiple sessions same week after streak increment: streak unchanged

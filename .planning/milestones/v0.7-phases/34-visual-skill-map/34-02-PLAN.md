      - Mastered: `Circle` stroke=gold (#fbbf24), fill=gold with lower opacity, strokeWidth=3
    - State icon overlay (positioned at top-right of circle):
      - Locked: Lock icon as SvgText emoji or small gray circle overlay
      - Mastered: Checkmark as SvgText emoji or small gold circle
      - Unlocked/in-progress: no overlay
    - Operation emoji: `SvgText` at center (cx, cy), fontSize ~16, textAnchor="middle", shows "+" for addition, "-" for subtraction (plain minus, not emoji)
    - Skill label: `SvgText` below circle (cx, cy + nodeRadius + 12), fontSize ~9, textAnchor="middle", abbreviated name (e.g. "Add 10", "Sub 20+"), fill from state-appropriate text color

    **Animations** (Reanimated):
    - Entrance: `useSharedValue(0)` for scale and opacity, triggered with `withDelay(entranceDelay, withSpring(1, {damping: 12, stiffness: 180}))` on mount.     - Performance: Only create animated shared values for nodes that need them (outer fringe). Static nodes use plain SVG props.

    **Fallback note:** If `useAnimatedProps` with `SVGAdapter` does not work reliably (the project hasn't tested this combination), fall back to wrapping node groups in absolute-positioned `Animated.View` components that handle opacity/scale, with static SVG inside.

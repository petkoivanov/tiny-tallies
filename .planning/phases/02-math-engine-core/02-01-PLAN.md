**src/services/mathEngine/seededRng.ts:**
Implement a mulberry32 seeded PRNG. This is a minimal deterministic PRNG (~15 lines) that produces reproducible sequences from an integer seed.

Export an interface and factory function:

```typescript
export interface SeededRng {
  /** Returns a float in [0, 1) */
  next(): number;
  /** Returns an integer in [min, max] inclusive */
  intRange(min: number, max: number): number;
}

export function createRng(seed: number): SeededRng {
  let state = seed | 0;

  function next(): number {
    state |= 0;
    state = (state + 0x6d2b79f5) | 0;
    let t = Math.imul(state ^ (state >>> 15), 1 | state);
    t = (t + Math.imul(t ^ (t >>> 7), 61 | t)) ^ t;
    return ((t ^ (t >>> 14)) >>> 0) / 4294967296;
  }

  return {
    next,
    intRange(min: number, max: number): number {
      return min + Math.floor(next() * (max - min + 1));
    },
  };
}
```

Use exactly this implementation. The mulberry32 algorithm is well-tested and produces good distribution for number selection.

**src/services/mathEngine/constraints.ts:**
Implement carry and borrow detection functions that iterate through all digit columns. These are used by the generator (Plan 02) to enforce template constraints.

```typescript
/** Returns true if adding a + b requires carrying in any column */
export function requiresCarry(a: number, b: number): boolean {
  while (a > 0 || b > 0) {
    if ((a % 10) + (b % 10) >= 10) return true;
    a = Math.floor(a / 10);
    b = Math.floor(b / 10);
  }
  return false;
}

/** Returns true if subtracting b from a requires borrowing in any column */
export function requiresBorrow(a: number, b: number): boolean {
  while (a > 0 || b > 0) {
    if ((a % 10) < (b % 10)) return true;
    a = Math.floor(a / 10);
    b = Math.floor(b / 10);
  }
  return false;
}
```

Also export a `validateOperands` helper that checks basic operand sanity:
```typescript
/** Validates that operands are non-negative integers */
export function validateOperands(a: number, b: number): void {
  if (!Number.isInteger(a) || !Number.isInteger(b)) {
    throw new Error(`Operands must be integers, got ${a} and ${b}`);
  }
  if (a < 0 || b < 0) {
    throw new Error(`Operands must be non-negative, got ${a} and ${b}`);
  }
}
```

Migration pattern for each file:
```typescript
// BEFORE:
import { colors, spacing } from '@/theme';
const styles = StyleSheet.create({
  container: { backgroundColor: colors.background },
});
export default function MyComponent() {
  return <View style={styles.container} />;
}

// AFTER:
import { useMemo } from 'react';
import { useTheme, spacing } from '@/theme';
export default function MyComponent() {
  const { colors } = useTheme();
  const styles = useMemo(() => StyleSheet.create({
    container: { backgroundColor: colors.background },
  }), [colors]);
  return <View style={styles.container} />;
}
```

Special case for Reanimated components (AvatarCircle, BadgeUnlockPopup, ConfettiCelebration):
- Colors used inside `useAnimatedStyle` callbacks reference the local `colors` variable via closure
- The useMemo pattern works identically -- just ensure `colors` is destructured before any animated style hooks

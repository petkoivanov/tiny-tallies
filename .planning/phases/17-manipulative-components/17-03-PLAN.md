**BarModelTypes.ts (~30 lines):**
- `interface SectionState { id: string; widthFraction: number; label: number | null; isUnknown: boolean; }`
  - `widthFraction`: proportion of total bar width (0.0 to 1.0). All sections' fractions sum to 1.0.
  - `label`: numeric value 0-999, or null if unlabeled
  - `isUnknown`: true if marked as "?" placeholder
- `interface BarModelProps { testID?: string; }`
- `type PartitionCount = 2 | 3 | 4`
- Color constants:
  - `BAR_PRIMARY = '#2DD4BF'` (teal per CONTEXT.md)
  - `BAR_FILL = 'rgba(45, 212, 191, 0.2)'`
  - `BAR_BORDER = '#14B8A6'`
  - `BAR_DIVIDER = '#FFFFFF'` (white divider handles for visibility on dark theme)
  - `BAR_UNKNOWN = '#F59E0B'` (amber for "?" sections — visually distinct)
- `const DIVIDER_WIDTH = 48` (touch target width for divider drag handles)
- `const BAR_HEIGHT = 80` (bar height in dp)
- `const SNAP_PERCENT = 10` (dividers snap to nearest 10% for child-friendly precision)
- `const NUMBER_PICKER_ITEM_HEIGHT = 48`
- `const NUMBER_PICKER_VISIBLE_ITEMS = 5`

**NumberPicker.tsx (~130 lines):**
A FlatList-based number picker wheel for values 0-999. No keyboard input.

- Props: `{ value: number; onChange: (value: number) => void; min?: number; max?: number; visible: boolean; onClose: () => void; }`
- Renders as a modal-like overlay (absolutely positioned within the bar model area, NOT a React Navigation modal — avoids gesture conflicts per project decision).
- **FlatList configuration:**
  - `data`: array `[min..max]` (default 0-999)
  - `snapToInterval={NUMBER_PICKER_ITEM_HEIGHT}` for wheel snap behavior
  - `decelerationRate="fast"`
  - `showsVerticalScrollIndicator={false}`
  - `getItemLayout` for O(1) scroll performance: `(_, index) => ({ length: ITEM_HEIGHT, offset: ITEM_HEIGHT * index, index })`
  - `initialScrollIndex={value - min}` to start at current value
  - `onMomentumScrollEnd`: compute selected index from `contentOffset.y / ITEM_HEIGHT`, call `onChange(min + index)`
- **Item rendering:** Each item is a `View` with `height: NUMBER_PICKER_ITEM_HEIGHT`, centered `<Text>` showing the number. Selected item (center of visible area) is larger/bolder. Use fading opacity for items away from center.
- **Visual:** Rounded container with `colors.surface` background, `BAR_BORDER` border. "Done" button at bottom to close. Dimmed backdrop behind picker.
- **Close behavior:** Tap backdrop or "Done" button calls `onClose()`. Value is committed immediately on scroll (no separate confirm needed beyond closing).
- **Accessibility:** `accessibilityLabel="Number picker"`. Each number item has `accessibilityLabel="{number}"`.
- Use `StyleSheet.create`.

  <files>src/components/manipulatives/BarModel/BarModel.tsx, src/components/manipulatives/BarModel/index.ts, src/components/manipulatives/index.ts</files>
  <action>
Create the BarModel manipulative (~300 lines).

**BarModel.tsx (~300 lines):**
- Uses `ManipulativeShell` with `count` = sum of all labeled section values, `countLabel` = "Total"
- State:
  - `partitionCount: PartitionCount | null` — null until user selects (2, 3, or 4)
  - `sections: SectionState[]` — array of section states with widthFractions summing to 1.0
  - `pickerTarget: { sectionId: string; currentValue: number } | null` — which section's picker is open
  - `barWidth: number` — measured from onLayout for pixel calculations
- **Initial state (no partition selected):** Show 3 preset buttons: "2 parts", "3 parts", "4 parts". Each is a large `Pressable` with visual preview of equal-width sections. Use teal color family. Touch targets: 48dp minimum.
- **After partition selected:** Render the bar model:
  - **Bar:** Horizontal container, `height: BAR_HEIGHT`, full available width (measured via `onLayout`). Background `BAR_FILL`. Border `BAR_BORDER`, `borderWidth: 2`, `borderRadius: layout.borderRadius.md`.
  - **Sections:** Each section renders as a `View` inside the bar with `width = widthFraction * barWidth`. Sections are side by side in a `flexDirection: 'row'` container.
    - Section background: `BAR_FILL` for normal, `BAR_UNKNOWN` with 0.3 alpha if `isUnknown`
    - Section content: Show `label` as centered `<Text>` if set, or "?" in `BAR_UNKNOWN` color if `isUnknown`, or empty
    - Tap section: If `isUnknown`, toggle unknown off. If labeled, open number picker. If empty, open action sheet with two options: "Set value" (opens picker) or "Mark as ?" (sets isUnknown=true)
    - Simpler approach: Tap section cycles through states: empty -> opens picker. Long press toggles "?" unknown. Or: Tap opens picker, and picker has a "?" button to mark unknown.
    - **Chosen UX:** Tap section opens picker overlay. Picker has a "?" button alongside numbers that marks the section as unknown and closes picker. If section already has "?", tapping it opens picker to replace with a number.
  - **Dividers:** Between adjacent sections, render a draggable divider handle.
    - Visual: Vertical line (`width: 3`, `backgroundColor: BAR_DIVIDER`) with a wider touch area (`width: DIVIDER_WIDTH`, transparent, centered on the line).
    - Drag: Use `Gesture.Pan()` on each divider. `onChange` updates the `widthFraction` of the two adjacent sections proportionally. The left section grows/shrinks as divider moves, and the right section compensates (widthFractions always sum to 1.0).
    - **Snap to percentage:** Divider positions snap to nearest `SNAP_PERCENT` (10%) of total bar width. This prevents children from creating arbitrarily thin sections. Computed: `roundedFraction = Math.round(newFraction * (100 / SNAP_PERCENT)) * SNAP_PERCENT / 100`. Clamp minimum section width to 10% (so no section disappears).
    - `onEnd`: Fire `triggerSnapHaptic()`.
    - Animated: Use `useSharedValue` for divider x-offset during active drag, commit to React state on end.
  - **Divider count:** For N sections, there are N-1 dividers. 2 parts = 1 divider, 3 parts = 2 dividers, 4 parts = 3 dividers.
- **Number picker integration:** When `pickerTarget` is set, render `<NumberPicker>` positioned near the tapped section (absolutely positioned overlay). On value change, update the section's `label`. On close, clear `pickerTarget`.
- **"?" unknown marking:** In the NumberPicker, add a "?" button at the top. Pressing it sets the section's `isUnknown = true`, clears any numeric `label`, and closes the picker.
- **Running count:** Sum of all numeric labels. Sections with "?" or no label contribute 0. Show in ManipulativeShell counter.
- **Reset:** Clear `partitionCount` back to null (return to preset selection). Clear all sections.
- **Accessibility:** Preset buttons: `accessibilityLabel="{N} equal parts"`. Sections: `accessibilityLabel="Section {n}, {label ?? 'empty'}{isUnknown ? ', unknown' : ''}"`. Dividers: `accessibilityLabel="Resize divider between section {n} and {n+1}"`.
- Use `StyleSheet.create`.

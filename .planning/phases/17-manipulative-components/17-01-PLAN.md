**TenFrame.tsx (~250 lines):**
- Uses `ManipulativeShell` as wrapper with `count` = total occupied cells across all frames
- State:
  - `cells: boolean[]` â€” array of booleans, starts as `new Array(CELLS_PER_FRAME).fill(false)`. When first frame is full, auto-extends to 20 cells (second frame).
  - `frameCount: number` â€” 1 initially, becomes 2 when all 10 cells of first frame are occupied
- `snapTargets` SharedValue: array of SnapTargets for all cells (10 or 20). Must be updated when frameCount changes.
- **Grid layout:** Each frame is a 2-row x 5-column grid of `SnapZone` components.
  - Top row: cells 0-4 (fill left to right)
  - Bottom row: cells 5-9
  - Use `flexDirection: 'row'` for each row, `flexWrap: 'nowrap'`
  - Each `SnapZone` has `id="cell-{index}"`, `isOccupied={cells[index]}`
  - When occupied, render an orange circular counter inside the SnapZone
- **Adding counters:** Tray at bottom with an orange counter source. User drags counter from tray; it snaps to nearest empty cell via DraggableItem + SnapZone pattern. On snap (`onSnap` callback): mark cell as occupied in state, update count. If target cell is already occupied, return to start (reject snap).
- **Removing counters:** Tap an occupied cell to remove the counter (fade-out animation using `FadeOut` from Reanimated layout animations). Update state to mark cell unoccupied.
- **Fill order enforcement:** When dragging, only register unoccupied cells as valid snap targets (filter `snapTargets` to exclude occupied cells). This naturally allows fill in any order (user decision says child CAN drag to any empty cell, not forced sequential).
- **Second frame auto-spawn:** When all 10 cells of first frame are occupied (`cells.slice(0, 10).every(Boolean)`), set `frameCount = 2` and extend cells array to 20. - **Reset:** `onReset` sets `cells` back to `new Array(CELLS_PER_FRAME).fill(false)`, `frameCount = 1`
- **SnapZone remeasurement:** When frameCount changes, the grid container's key changes, causing remount. All SnapZones call `onMeasured` again with new positions. Use `useSharedValue` for snap targets and update via `onMeasured` callback that builds the targets array.

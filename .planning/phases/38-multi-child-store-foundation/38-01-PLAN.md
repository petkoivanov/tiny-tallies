Create `src/store/helpers/childDataHelpers.ts`:

1. Define the `ChildData` interface with all 18 per-child fields. Use the types from existing slices:
   - From childProfileSlice: childName (string), childAge (number), childGrade (number), avatarId (AllAvatarId | null), frameId (FrameId | null), themeId (ThemeId), tutorConsentGranted (boolean)
   - From skillStatesSlice: skillStates (Record<string, SkillState>)

   - From achievementSlice: earnedBadges (Record<string, EarnedBadge>)
   - From challengeSlice: challengeCompletions (Record<string, ChallengeCompletion>), challengesCompleted (number)

5. Define `DEFAULT_CHILD_DATA` with sensible defaults (xp: 0, level: 1, themeId: 'dark', empty collections).

6. Define `createDefaultChildData(profile: { childName: string; childAge: number; childGrade: number; avatarId: AllAvatarId | null; skillStates?: Record<string, SkillState> }): ChildData` -- merges profile info with defaults.

7. Also export `NewChildProfile` type: `{ childName: string; childAge: number; childGrade: number; avatarId: AllAvatarId | null }`.

  <action>
Create a full-screen confetti particle overlay using react-native-reanimated.

Component: ConfettiCelebration (no props needed — auto-plays on mount).

Implementation approach — simple particle system with reanimated:

1. Define a PARTICLE_COUNT constant (20-30 particles).

2. Create a Particle sub-component that renders a small colored square/rectangle:
   - Random initial x position across screen width (use Dimensions.get('window'))
   - Animate translateY from -20 to screen height + 20 using withTiming over 2000-3000ms with random delay (0-800ms) for staggered effect
   - Animate opacity: start at 1, fade to 0 in last 500ms using withTiming
   - Animate rotation using withRepeat + withTiming for gentle spinning
   - Random size between 8-14px
   - Random color from celebration palette: [colors.correct, colors.primaryLight, colors.primary, '#fbbf24' (amber), '#f472b6' (pink), '#34d399' (emerald)]
   - Each particle uses useAnimatedStyle for translateY, opacity, and rotation

3. Container: absolute positioned View covering full screen with pointerEvents="none" so it doesn't block touch on buttons underneath.

4. Use useEffect on mount to trigger all particle animations simultaneously (each with its own random delay for stagger).

5. Keep total animation duration around 2.5 seconds. After animations complete, the component can remain mounted — particles will be invisible (opacity 0).

Use StyleSheet.create for styles. Keep component under 120 lines. Do NOT use any external confetti library — build with reanimated primitives.

**Step A: Create formatAsMultiSelect (NEW -- answer format pipeline extension)**

Create `src/services/mathEngine/answerFormats/multiSelect.ts`:
- Export `formatAsMultiSelect(problem: Problem, seed: number): MultiSelectPresentation`
- Implementation per Research Pattern 2:
  - Assert answer.type === 'multi_select', throw otherwise
  - Extract correctValues from answer.values
  - Build distractor pool from problem.operands (skip values already in correctValues)
  - Take up to 2 distractors from operands
  - Build ChoiceOption[] array: correctValues + distractors
  - Shuffle deterministically using createRng(seed) and shuffleArray
  - Compute correctIndices by finding positions of correct values in shuffled array
  - Return { problem, format: 'multi_select', options: shuffled, correctIndices }

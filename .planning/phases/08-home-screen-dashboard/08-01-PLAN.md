<task type="auto">
  <name>Task 1: Rewrite HomeScreen with dashboard layout</name>
  <files>src/screens/HomeScreen.tsx</files>
  <action>
Rewrite HomeScreen.tsx from the current centered placeholder into a top-heavy dashboard layout. Use StyleSheet.create for all styles (per CLAUDE.md). Import from existing services and store -- do NOT create new files.

**Layout structure (top to bottom):**

1. **Profile Section (top area):**
   - Avatar: render the child's avatar emoji from AVATARS constant (look up by avatarId from store, fallback to DEFAULT_AVATAR_ID) inside a circular View (colors.surface background, ~80dp diameter, borderRadius round)
   - Hero greeting: "Hi, [childName]!" in Lexend bold, fontSize display. If childName is null, show "Hi, Mathematician!"
   - Level badge: "Level [level]" text beneath the greeting in primaryLight color, fontSize xl

2. **Stats Section (middle area):**
   - **XP Progress Bar:**
     - Use calculateLevelFromXp(xp) from levelProgression.ts to get xpIntoCurrentLevel and xpNeededForNextLevel
     - Show text: "[xpIntoCurrentLevel] / [xpNeededForNextLevel] XP" above the bar
     - Bar container: full width, height ~12dp, borderRadius round, backgroundColor surface
     - Bar fill: width = (xpIntoCurrentLevel / xpNeededForNextLevel) * 100%, backgroundColor primary, borderRadius round
     - Minimum fill width of 4dp when xpIntoCurrentLevel > 0 so the bar is always visible
   - **Streak Display:**
     - Use isSameISOWeek from weeklyStreak.ts to derive practicedThisWeek: compare lastSessionDate (parse to Date) with new Date()
     - If lastSessionDate is null, practicedThisWeek = false
     - Show streak icon (Flame from lucide-react-native), streak count text ("[weeklyStreak] week streak")
     - If practicedThisWeek is true, show a checkmark icon (Check from lucide-react-native) and use colors.correct for the check
     - If practicedThisWeek is false AND weeklyStreak > 0, show gentle nudge: "Ready to keep your streak going?" in textSecondary (NOT punitive language)
     - If weeklyStreak is 0, show: "Start your streak!" in textSecondary

3. **Start Practice Button (bottom, fixed):**
   - Position the button area at the bottom using flex layout (content section gets flex:1, button section is fixed)
   - Large Pressable: full width minus horizontal padding, minHeight 56dp (larger than standard 48dp for primary CTA), backgroundColor primary, borderRadius lg
   - Pressed state: backgroundColor primaryDark
   - Text: "Start Practice" in Lexend semiBold, fontSize lg, centered
   - accessibilityRole="button", accessibilityLabel="Start Practice"
   - onPress: navigate('Session', { sessionId: String(Date.now()) }) -- same as current

**State access:**
- Use useAppStore with individual selectors to minimize re-renders: useAppStore(state => state.xp), etc. (per Zustand atomic state best practice)
- Import: colors, spacing, typography, layout from '@/theme'
- Import: useSafeAreaInsets from 'react-native-safe-area-context'
- Import: useNavigation from '@react-navigation/native'
- Import: Flame, Check from 'lucide-react-native'
- Import: AVATARS, DEFAULT_AVATAR_ID from store/constants/avatars
- Import: calculateLevelFromXp from services/gamification/levelProgression
- Import: isSameISOWeek from services/gamification/weeklyStreak

**Style notes:**
- Use paddingTop: insets.top, paddingBottom: insets.bottom on outer container
- Horizontal padding: spacing.lg on content areas
- Gap between sections: spacing.xl
- All text uses Lexend font variants from typography.fontFamily
- Dark theme: background color is colors.background

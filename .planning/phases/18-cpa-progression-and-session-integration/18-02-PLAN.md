Purpose: Pictorial mode (moderate mastery) shows static visual aids between the problem text and answer buttons. Each diagram visualizes the problem operands in the style of its manipulative type -- dot groups for counters, grid cells for ten frames, block diagrams for base-ten, etc.
Output: 6 per-type SVG renderers + 1 dispatcher + barrel export + tests, all under src/components/session/pictorial/.

**generators.ts:** Create 6 generator functions using the construction-from-answer pattern. All use `numericAnswer`.

`generateFoilExpansion(template, rng)`:
- Pick a, b in intRange(-5, 5), re-roll if 0. Pick evaluation x in intRange(1, 4).
- answer = (x + a) * (x + b). forgotMiddle = x*x + a*b (F+L only, missing OI middle terms).
- questionText: "Expand (x {signedConst(a)})(x {signedConst(b)}) and evaluate at x = {x}."
- operands: [a, b, x, forgotMiddle]. Use a helper `signedConst(n)` that returns e.g., "+ 3" or "- 2".
- return numericAnswer(answer).

`generatePolyEvaluation(template, rng)`:
- Pick coeffA in intRange(1, 5), coeffB in intRange(-8, 8), coeffC in intRange(-10, 10), evalX in intRange(-3, 5).
- answer = coeffA * evalX^2 + coeffB * evalX + coeffC.
- questionText: "Evaluate {coeffA}x\u00B2 {signedTerm(coeffB, 'x')} {signedConst(coeffC)} at x = {evalX}."
- operands: [coeffA, coeffB, coeffC, evalX].
- return numericAnswer(answer).

`generateGcfFactoring(template, rng)`:
- Pick gcf in intRange(2, 5), innerA in intRange(1, 4), innerB in intRange(1, 6).
- Correct factored form label: "{gcf}({innerA}x + {innerB})" (use signedConst for innerB).
- Distractors (3 wrong labels with unique numeric values):
  - value=1, label: partial GCF form (e.g., "{gcf/2 or 2}({gcf*innerA/2}x + {gcf*innerB/2})")
  - value=2, label: unfactored-looking form (e.g., "{gcf*innerA}x + {gcf*innerB}" -- not factored)
  - value=3, label: wrong constant (e.g., "{gcf}({innerA}x + {gcf*innerB})" -- forgot to divide)
- correctAnswer: numericAnswer(0). Store MC option data in operands: [gcf, innerA, innerB].
- questionText: "Factor {gcf*innerA}x + {gcf*innerB}. Choose the correct factored form."
- CRITICAL: The template for gcf_factoring must set `domainConfig: { type: 'gcf_factoring', expressionMC: true }` so `selectAndFormatAnswer` can route to MC always.

`generateDiffOfSquares(template, rng)`:
- Pick b in intRange(2, 8).
- Correct: "(x + {b})(x \u2212 {b})". Label = "(x + {b})(x - {b})".
- Distractors: value=1 "(x + {b})\u00B2", value=2 "(x - {b})\u00B2", value=3 "(x + {b*b})(x - 1)".
- correctAnswer: numericAnswer(0).
- operands: [b, 0, 0] (pad to 3 for consistent length).
- questionText: "Factor x\u00B2 \u2212 {b*b}. Choose the correct factored form."

`generateCombinedOps(template, rng)`:
- Pick a, b, c (small integers). Form: simplify (x + a)(x + b) + c and evaluate at x = evalPoint.
- answer = (evalPoint + a) * (evalPoint + b) + c.
- operands: [a, b, c, evalPoint].
- questionText describes the combined expression to simplify and evaluate.

`generateWordProblem(template, rng)`:

- This generator is invoked by the word problem system's prefix mode.

**generators.ts:** Create 5 generator functions using the construction-from-answer pattern. All use `numericAnswer`. Copy the SUPERSCRIPTS map from exponents/generators.ts for exp_evaluate.

`generateExpEvaluate(template, rng)`:
- Pick base = rng.intRange(2, 6), exp = rng.intRange(2, 4).
- answer = Math.pow(base, exp). Max: 6^4 = 1296 (under 2000).
- wrongMultiply = base * exp (linear thinking bug).
- wrongPlusOne = Math.pow(base, exp + 1) -- but cap: if > 1999, use Math.pow(base, exp - 1).
- sup = SUPERSCRIPTS[exp] or `^${exp}`.
- questionText: `Evaluate ${base}${sup}.`
- operands: [wrongMultiply, wrongPlusOne, answer].
- return numericAnswer(answer).

`generateGrowthFactor(template, rng)`:
- Pick initial = rng.intRange(2, 10), factor = rng.intRange(2, 4) (doubles/triples/quadruples), periods = rng.intRange(2, 4).
- answer = initial * Math.pow(factor, periods). Max: 10 * 4^4 = 2560 -- to stay under 2000, cap: if answer >= 2000, reduce periods or initial. Safe approach: initial in [2, 5], factor in [2, 3], periods in [2, 3]. Max: 5 * 3^3 = 135. Or allow initial up to 10 with factor=2, periods=3 -> 10*8=80. With factor=3, periods=3 -> 10*27=270. All safe.
- wrongAdd = initial + factor * periods (linear thinking).
- wrongOneLess = initial * Math.pow(factor, periods - 1) (off-by-one period).
- factorWord = factor === 2 ? 'doubles' : factor === 3 ? 'triples' : 'quadruples'.
- questionText: `A colony starts with ${initial} bacteria and ${factorWord} every hour. How many after ${periods} hours?`
- operands: [wrongAdd, wrongOneLess, answer].
- return numericAnswer(answer).

`generateDecayFactor(template, rng)`:
- Pick initial from [64, 128, 256, 512, 1024] (powers of 2 guarantee integer halving).
- periods = rng.intRange(1, 4). answer = initial / Math.pow(2, periods). Always integer.
- wrongDouble = initial * Math.pow(2, periods) -- but cap: if > 1999, use initial * 2 instead.
- wrongOneLess = initial / Math.pow(2, Math.max(1, periods - 1)) (off-by-one).
- questionText: `A substance starts at ${initial}g and loses half its mass every year. How many grams remain after ${periods} years?`
- operands: [wrongDouble, wrongOneLess, answer].
- return numericAnswer(answer).

`generateDoublingTime(template, rng)`:
- Pick initial = rng.intRange(10, 100) (step by 10 for clean numbers: rng.intRange(1, 10) * 10).
- doublingPeriod = rng.intRange(2, 4). totalTime = doublingPeriod * rng.intRange(2, 3) (ensures totalTime/doublingPeriod is integer 2 or 3).
- numDoublings = totalTime / doublingPeriod.
- answer = initial * Math.pow(2, numDoublings). Max: 100 * 2^3 = 800. Safe.
- wrongLinear = initial * 2 * numDoublings (linear doubling, not exponential).
- wrongOneLess = initial * Math.pow(2, numDoublings - 1).
- questionText: `An investment starts at $${initial} and doubles every ${doublingPeriod} years. What is it worth after ${totalTime} years?`
- operands: [wrongLinear, wrongOneLess, answer].
- return numericAnswer(answer).

    **types.ts** (~40 lines): Define interfaces consumed by Plan 02 components:
    - `SnapTarget`: `{ id: string; cx: number; cy: number; width: number; height: number }` -- center-based coordinates for snap zones
    - `DraggableItemProps`: `{ id: string; snapTargets: SharedValue<SnapTarget[]>; snapThreshold?: number; onSnap?: (itemId: string, targetId: string) => void; onTap?: (itemId: string) => void; onDragStart?: (itemId: string) => void; enabled?: boolean; style?: StyleProp<ViewStyle>; accessibilityLabel: string; children: React.ReactNode }` -- Import SharedValue from react-native-reanimated, StyleProp/ViewStyle from react-native

    - `AnimatedCounterProps`: `{ value: number; label?: string; style?: StyleProp<ViewStyle> }`
    - `ResetConfig`: `{ animate?: boolean; staggerMs?: number }` -- controls reset animation style

    **snapMath.ts** (~60 lines): Pure worklet-annotated functions. Write tests first (TDD).
    - `findNearestSnap(x, y, targets, threshold)`: Returns nearest SnapTarget within threshold distance (Euclidean distance from point to target center), or null. Iterates targets array, tracks minimum distance. Must have `'worklet';` directive.
    - `isInsideZone(x, y, zone)`: Returns boolean for axis-aligned bounding box containment check (inclusive of edges). Half-width/half-height from center. Must have `'worklet';` directive.

    - `SNAP_SPRING_CONFIG`: `{ damping: 20, stiffness: 300, mass: 0.8, overshootClamping: true }` -- fast, no bounce (precision placement per Pitfall 5)

    - `RESET_SPRING_CONFIG`: `{ damping: 18, stiffness: 180, mass: 1, overshootClamping: true }` -- gentle cascade
    - `COUNTER_POP_CONFIG`: `{ damping: 8, stiffness: 400, mass: 0.5 }` -- pop-in effect
    - `SNAP_THRESHOLD = 50` -- pixels, snap distance
    - `MAX_OBJECTS = 30` -- object cap per research requirements
    - `DRAG_SCALE = 1.08` -- scale during drag
    - `DRAG_OPACITY = 0.85` -- opacity during drag
    - `RESET_STAGGER_MS = 30` -- stagger delay per item on reset

    **haptics.ts** (~20 lines): RN-thread haptic functions (called via scheduleOnRN from worklets):
    - `triggerSnapHaptic()`: Calls `Haptics.impactAsync(ImpactFeedbackStyle.Light)` -- per locked decision
    - `triggerGroupHaptic()`: Calls `Haptics.notificationAsync(NotificationFeedbackType.Success)` -- per locked decision
    - Import from `expo-haptics`. These are regular functions (NOT worklets). They are called via `scheduleOnRN(triggerSnapHaptic)` from worklet onEnd handlers.

- Impact: Redundant header row in session view. Inner shell works correctly.
- Affected: SESS-01, MANIP-08
- Origin: Phase 18 (pre-existing)

**INT-02: Missing Guided Step Subtraction Resolvers** (Low)
- `guidedSteps.ts` has resolvers for 7 operation+manipulative pairs. Missing: base_ten_blocks+subtraction, number_line+subtraction, bar_model+subtraction.
- Impact: `getNextGuidedStep` returns null → no glow, no hint text for those combinations. Graceful degradation (no crash).
- Affected: POL-01 (partial coverage)
- Fix: Add 3 subtraction resolvers to guidedSteps.ts
